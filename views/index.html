Timestamp Microservice

Example Usage

Base url: 
          https://simple-timestamp-service.herokuapp.com
Valid date 
          GET /api/timestamp/2020-12-30
        
{
  "unix": 1609286400000,
  "utc": "Wed, Dec 30, 2020, 01:00:00 GMT+1"
}
Unix timestamp 
          GET /api/timestamp/1451001600000
        
{
  "unix": 1451001600000,
  "utc": "Fri, Dec 25, 2015, 01:00:00 GMT+1"
}
Empty parameter → current time 
          GET /api/timestamp
        
{
  "unix": 1611947123417,
  "utc": "Fri, Jan 29, 2021, 20:05:23 GMT+1"
}
Invalid date 
          GET /api/timestamp/hello-guys
        
{
  "error": "Invalid Date"
}
